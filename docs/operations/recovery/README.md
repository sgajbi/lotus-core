# Runtime Recovery

This folder owns executable recovery procedures for Lotus Core deployable boundaries. Each runbook
must identify the interrupted owner, durable recovery boundary, exact invariants, generated
evidence, and enforcement command.

## Procedures

- [Portfolio derived-state interruption recovery](./portfolio-derived-state-interruption.md)

Transaction processing cutover and interruption recovery remains in the
[Transaction Processing Cutover Runbook](../Transaction-Processing-Cutover-Runbook.md) until that
broader runbook is reorganized under its owning capability.
